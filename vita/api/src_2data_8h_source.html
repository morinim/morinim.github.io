128  // Data are stored in three datasets:
129  // * a training set used directly for learning;
130  // * a validation set for controlling overfitting and measuring the
131  // performance of an individual;
132  // * a test set for a forecast of how well an individual will do in the
133  // real world.

135  // model. We don't stop at the validation step because we've iteratively
