Detailed Description

template<class T, bool S>
class vita::basic_reg_lambda_f< T, S >

Transforms individual to a lambda function for regression.

Template Parameters
Ttype of individual
Sstores the individual inside vs keep a reference only. Sometimes we need an autonomous lambda function that stores everything it needs inside (slow but will survive the death of the the program it's constructed on); sometimes we need speed, not persistence
