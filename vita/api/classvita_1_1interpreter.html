Detailed Description

template<class T>
class vita::interpreter< T >

A template specialization of the core_interpreter class.

Template Parameters
Tthe type of individual used.
Note
This class would like to be a generic interpreter<T> implementation, but interpreter and individual are strongly coupled objects: the interpreter must be build around the peculiarities of the specific individual class. So don't expect to use this template for T different from i_mep without some modifications.
This is an example of dependency injection via constructor injection: the class gets the dependencies as an argument to its constructor. It saves the reference to dependent classes as private variables. All the dependencies are truly visible with constructor signature and cyclic dependencies don't happen because of the well-defined order of instantiation.
See also
