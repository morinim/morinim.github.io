Detailed Description

template<class T>
class vita::sum_of_errors_evaluator< T >

An evaluator to minimize the sum of some sort of error.

This class models the evaluators that will drive the evolution towards the minimum sum of some sort of error.

See also
mse_evaluator, mae_evaluator, rmae_evaluator.
