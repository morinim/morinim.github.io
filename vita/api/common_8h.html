#define DISALLOW_COPY_AND_ASSIGN (   Class)
Value:
Class(const Class &) = delete; \
Class &operator=(const Class &) = delete

A macro to disallow the copy constructor and operator= functions.

A good alternative is boost::noncopyable but that gives some false positive with g++ -Weffc++ so, for now, we keep the macro.

See also
http://stackoverflow.com/q/7823990/3235496 and http://stackoverflow.com/q/1454407/3235496 for further details.
