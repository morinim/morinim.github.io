const symbol & vita::symbol_set::collection::sum_container::roulette ( ) const
Returns
a random symbol from the container.

Probably the fastest way to produce a realization of a random variable X in a computer is to create a big table where each outcome i is inserted a number of times proportional to P(X=i).

Two fast methods are described in "Fast Generation of Discrete Random Variables" (Marsaglia, Tsang, Wang). Also std::discrete_distribution seems quite fast.

Anyway we choose the "roulette algorithm" because it's very simple.
