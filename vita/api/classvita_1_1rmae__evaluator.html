Detailed Description

template<class T>
class vita::rmae_evaluator< T >

Evaluator based on the mean of relative differences.

This evaluator will drive the evolution towards the minimum sum of relative differences between target values and actual ones:

This is similar to mae_evaluator but here we sum the relative errors. The idea is that the absolute difference of 1 between 6 and 5 is more significant than the same absolute difference between 1000001 and 1000000. The mathematically precise way to express this notion is to calculate the relative difference.
