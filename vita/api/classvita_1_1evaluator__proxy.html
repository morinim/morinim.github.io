Detailed Description

template<class T>
class vita::evaluator_proxy< T >

Template Parameters
Tthe type of individual used

Provides a surrogate for an evaluator to control access to it. The reason for controlling access is to cache fitness scores of individuals. evaluator_proxy uses an ad-hoc internal hash table (cache).
